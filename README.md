# PDFMASTER

📄 PDF Converter Web App
🚀 Overview

A modern client-side PDF utility web app built using HTML, CSS, and JavaScript.
All processing happens locally in the browser — no server required.

✨ Features
🖼️ IMG → PDF

Convert JPG, PNG, and WebP images into a downloadable PDF.

📚 Merge PDF

Combine multiple PDF files into one.

✂️ Split PDF → Images

Extract each PDF page as a PNG image and download as a ZIP.

➕ Add Pages to PDF

Insert blank or image-based pages into an existing PDF.

✍️ Add Signature to PDF

Draw or upload a signature and place it onto a PDF.

🛠️ Tech Stack
pdf-lib → Create, merge, modify PDFs
PDF.js → Render PDF pages
JSZip → Generate ZIP files
FileSaver.js → Download files
