FAQ

Common Questions

Are my files uploaded to any server? +
No. Every operation happens entirely in your browser using JavaScript. Your files are never sent to any server.
What image formats are supported for conversion? +
JPG, JPEG, PNG, WebP, GIF, and BMP images are all supported for the Image to PDF tool.
Is there a file size limit? +
There is no hard limit enforced by DocForge. The limit is your browser's available memory — typically this is several hundred MB comfortably.
Can I reorder images before converting to PDF? +
Yes! After uploading images you'll see a preview list where you can drag to reorder or remove any image before generating the PDF.
